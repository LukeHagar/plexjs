# Format

The output format for the image; defaults to jpg

## Example Usage

```typescript
import { Format } from "@lukehagar/plexjs/models/operations";

let value: Format = Format.Jpg;
```

## Values

| Name   | Value  |
| ------ | ------ |
| `Jpg`  | jpg    |
| `Jpeg` | jpeg   |
