# AutoSelectSubtitle

The auto-select subtitle mode (0 = Manually selected, 1 = Shown with foreign audio, 2 = Always enabled)

## Example Usage

```typescript
import { AutoSelectSubtitle } from "@lukehagar/plexjs";

let value: AutoSelectSubtitle = AutoSelectSubtitle.One;
```

## Values

| Name   | Value  |
| ------ | ------ |
| `Zero` | 0      |
| `One`  | 1      |
