# WriteMessageLevel

An integer log level to write to the PMS log with.
  - 0: Error
  - 1: Warning
  - 2: Info
  - 3: Debug
  - 4: Verbose


## Example Usage

```typescript
import { WriteMessageLevel } from "@lukehagar/plexjs/sdk/models/operations";

let value: WriteMessageLevel = WriteMessageLevel.Four;
```
