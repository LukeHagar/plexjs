# LineupType

- `-1`: N/A
- `0`: Over the air
- `1`: Cable
- `2`: Satellite
- `3`: IPTV
- `4`: Virtual


## Example Usage

```typescript
import { LineupType } from "@lukehagar/plexjs/sdk/models/shared";

let value: LineupType = LineupType.Two;
```
